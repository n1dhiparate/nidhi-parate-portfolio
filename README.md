## 🚀 Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn package manager

### Installation

```bash
# Clone the repository
git clone https://github.com/n1dhiparate/nidhi-parate-portfolio.git

# Navigate to the project directory
cd nidhi-parate-portfolio

# Install dependencies
npm install
# or
yarn install
```

### Development

```bash
# Start the development server
npm run dev
# or
yarn dev
```

The website will be available at `http://localhost:3000` (or the port displayed in your terminal).

### Build

```bash
# Create a production build
npm run build
# or
yarn build
```
